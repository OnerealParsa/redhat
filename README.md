## معرفی پروژه

این پروژه یک بازی متنی شبیه‌سازی شده بر اساس داستان شنیل قرمزی است که در آن بازیکن باید از میان جنگل عبور کند و به خانه مادربزرگ برسد بدون اینکه گرگ او را بگیرد.

### زبان برنامه‌نویسی
C++ (استاندارد C++17)

### ساختارهای داده استفاده شده
- **Graph**: نمایش نقشه بازی و مسیرها
- **Hash Table**: ذخیره اطلاعات کاربران
- **BST (درخت جستجوی دودویی)**: ذخیره و جستجوی امتیاز کاربران
- **Max Heap**: استخراج برترین بازیکن
- **Stack**: پیاده‌سازی قابلیت Undo

### الگوریتم‌ها
- **Dijkstra**: محاسبه کوتاه‌ترین مسیر بازیکن تا خانه
- **BFS**: محاسبه کوتاه‌ترین مسیر گرگ تا بازیکن
- **A***: مسیر یابی با تابع هیوریستیک (بخش امتیازی)

### نحوه کامپایل و اجرا

```bash
make
./hoodquest
```

یا به صورت دستی:

```bash
g++ -std=c++17 -O2 -o hoodquest main.cpp graph.cpp hash_table.cpp bst.cpp max_heap.cpp game_map.cpp game.cpp user_manager.cpp
./hoodquest
```

### نحوه بازی
- `R` = شنیل قرمزی (بازیکن)
- `G` = گرگ
- `V` = خانه مادربزرگ
- `T` = تله
- `#` = دیوار
- `.` = خانه خالی

حرکت با وارد کردن مختصات (ردیف ستون) انجام می‌شود.
- برای Undo: `undo`
- برای خروج: `quit`

### سیستم امتیازدهی
| عمل | امتیاز |
|------|--------|
| حرکت طبق پیشنهاد Dijkstra | +3 |
| حرکت غیر از پیشنهاد | +1 |
| استفاده از Undo | -2 |
| رسیدن به خانه | +5 |
| امتیاز اضافه برد | +5 |

### ساختار فایل‌ها
| فایل | توضیح |
|------|--------|
| `main.cpp` | نقطه ورود برنامه، منوی اصلی و مدیریت بازی |
| `graph.h / graph.cpp` | پیاده‌سازی گراف، Dijkstra، BFS و A* |
| `hash_table.h / hash_table.cpp` | پیاده‌سازی Hash Table برای ذخیره کاربران |
| `bst.h / bst.cpp` | پیاده‌سازی BST برای جستجوی امتیاز کاربران |
| `max_heap.h / max_heap.cpp` | پیاده‌سازی Max Heap برای برترین بازیکن |
| `stack.h` | پیاده‌سازی Stack (template) برای Undo |
| `game_map.h / game_map.cpp` | پیاده‌سازی نقشه بازی و تولید تصادفی محیط |
| `game.h / game.cpp` | منطق بازی، امتیازدهی و حرکات بازیکن و گرگ |
| `user_manager.h / user_manager.cpp` | مدیریت کاربران، احراز هویت و ذخیره‌سازی |
| `Makefile` | فایل ساخت پروژه |
